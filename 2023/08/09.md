## わかったこと
* 戻り値
  Rubyにおいて、戻り値はメソッド内で最後に評価されたものが返ってくる
  JSなどのように`return`の記載は必要ない

* puts, print,p
  * puts   
    引数の文字列を改行して表示する。
  * print
    改行せずに表示する
  * p
    デバッグ用

* ハッシュ
  以下のように指定できる
  ```
  sample = {
    'name' => 'john'
    'age' => 13
  }
  sample['name']

  human = {
    name: 'ben'
    age: '11'
  }
  human[:name]
  ```
* each, for
  each
  ```
  配列やハッシュ.each do |var|
   # 処理
  end
  ```
  for
  ```
  for var in 配列やハッシュ do

  end
  ```
